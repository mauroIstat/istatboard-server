Installation
Clone repo

# clone the repo
$ git clone https://github.com/mecdcme/istatboard-server.git istatboard-server

# go into app's directory
$ cd istatboard-server 

# install app's dependencies
$ npm install

Usage

# serve with hot reload at localhost:8080.
$ node server.js
